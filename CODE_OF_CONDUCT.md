# Code of Conduct

ForestMate is an open project for hiking safety, forest public data, and
responsible AI-assisted outdoor services. We want contributors, users, reviewers,
and public-sector collaborators to be able to participate in a respectful and
productive environment.

## Expected Behavior

- Be respectful, patient, and constructive.
- Assume good intent, but take responsibility for the impact of your words and
  actions.
- Welcome contributors with different backgrounds, experience levels, languages,
  and accessibility needs.
- Keep discussions focused on the project, the evidence, and the user impact.
- Treat safety, location, emergency, and public-data topics with extra care.

## Unacceptable Behavior

- Harassment, threats, insults, personal attacks, or discriminatory language.
- Sexualized language or imagery in project spaces.
- Publishing another person's private information, including precise location
  data, contact details, credentials, or API keys.
- Dismissing safety, privacy, or accessibility concerns without review.
- Repeatedly derailing issues, pull requests, or discussions after maintainers
  ask for focus.

## Reporting

If you see behavior that violates this code of conduct, contact the project
maintainers through an appropriate private channel. If the report involves a
security issue, private location data, credentials, or potential harm to users,
use GitHub's private vulnerability reporting or another private contact method
from the maintainer profile rather than a public issue.

Reports should include:

- What happened and where it happened.
- The people or accounts involved.
- Any relevant links, screenshots, or timestamps.
- Whether immediate moderation, privacy removal, or security handling is needed.

Maintainers may remove content, moderate discussions, close issues, limit
participation, or take other actions needed to protect the community and users.

## Scope

This code applies in ForestMate project spaces, including GitHub issues, pull
requests, discussions, release comments, documentation review, and any community
channel operated for this repository.
